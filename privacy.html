我們的隱私承諾

本隱私聲明描述了我們收集的個人資訊類型、收集目的、披露方式，以及您在訪問或使用我們的功能、服務、活動（統稱 “服務”）時享有的選擇權。我們同樣認識到需要為訪問和 / 或使用我們服務的兒童提供額外保護。如需瞭解針對兒童的隱私保護措施，請參閱本聲明附件《兒童隱私補充聲明》。

一、本隱私聲明適用範圍

本隱私聲明適用於由 C Dessert Limited（以下簡稱「本公司」）所運營的移動應用程式（App）及其相關服務。

請你在使用本 App 前仔細閱讀本聲明。你下載、註冊或繼續使用本 App，即表示你已閱讀、理解並同意本隱私聲明的全部內容。

本隱私聲明不適用於人力資源數據或 “招聘” 相關網站。

二、我們收集的個人資訊

（一）您直接提供的資訊

當您創建帳戶、進行購買、填寫表單、完成調查、通過我們的服務提交或與我們溝通、參與競賽、抽獎或促銷活動、訂閱我們的行銷通訊、請求客戶支持或通過其他方式與我們溝通時，我們會直接從您那裏收集個人資訊。我們直接收集的個人資訊類型包括：

1.聯繫資訊和識別字：例如您的姓名、電子郵箱地址、電話號碼、郵寄地址、帳號 ID 等。

2.內容和文件：您分享或上傳的數據和內容，例如照片（包括照片和內容中包含的任何元數據）。

3.支付資訊：如果您從我們這裏購買產品或服務，我們會與第三方支付處理商合作處理您的支付資訊，如信用卡號或金融帳戶資訊。

4.人口統計資訊：您在註冊帳戶或填寫調查時提供的資訊，例如年齡和 / 或出生日期、性別、居住國家 / 地區、愛好和偏好。

5.位置數據：在某些服務中，我們可能收集大致位置資訊，如州、城市或您的地址。

（二）我們自動收集的資訊

當您使用我們的服務時，我們會自動收集某些資訊，例如：

1.識別字和設備資訊：例如設備類型、操作系統類型、流覽器類型、設備 ID、互聯網協議（IP）地址，以及其他軟體的類型、版本、語言、設置和配置。

2.通過 Cookie 和類似跟蹤技術收集的資訊：我們使用 Cookie 和類似技術（如像素）存儲和檢索數據，以收集有關您與服務和行銷通訊互動的資訊。這些技術幫助我們改進服務和行銷通訊、個性化您的體驗，並分析您與我們的互動。有關我們使用的 Cookie 和跟蹤技術以及您可享有的選擇權的更多資訊，請參閱我們的《Cookie 政策》。

3.使用數據：例如您來自哪個網站、使用我們服務的時間和時長、訪問的頁面和內容以及使用的功能。

4.位置數據：在某些服務中，我們可能收集大致位置資訊，例如從您的 IP 地址獲取的位置資訊。

5.客戶支持中心通訊：在美國，撥打我們客戶支持中心的電話可能會被監控和保留，用於品質保證和培訓目的。接通前會通知您。與客戶支持的短信和聊天記錄也可能為品質保證和培訓目的被監控和保留。

6.安全攝像頭錄影：我們的自動零售設備（也稱為自動售貨機）配備了攝像頭設備，可能會錄製視頻片段以防止未授權訪問。

7.購買交易資訊：如果您在我們的服務或自動售貨機上進行購買，我們會收集您的交易資訊，包括產品詳情、購買價格和交易日期。

（三）我們創建或生成的資訊

我們可能會從收集到的關於您的數據中推導其他資訊，並可能將其與其他個人資訊相結合，生成關於您可能的偏好或其他特徵的資訊（“推斷資訊”）。例如，我們會從您的 IP 地址推斷您的大致位置。

（四）我們從其他來源獲取的資訊

我們從其他來源獲取資訊，包括：

1.非本公司帳戶運營商；

2.移動平臺：當您下載我們的應用程式和 / 或進行應用內購買時，我們會從應用商店和其他移動平臺提供商那裏收到關於您的資訊，這些資訊僅限於用戶名或螢幕名稱和 / 或設備 ID 以及購買事實，除非您授權移動平臺提供商向我們提供額外資訊；

3.合作商：在通過相關合作商完成購買的情況下，我們會收到訂單和客戶詳情；

4.服務提供商：服務提供商可能會收集上述任何類別的個人資訊，以便代表我們提供服務，包括幫助我們根據 IP 地址確定您設備的大致位置，為您提供更便捷的體驗，並提供額外的安全措施來保護我們的服務；

5.社交網路：如果相關資訊（包括您的姓名、螢幕名稱和您發佈的內容）對我們公開可用，我們可能會將此類數據用於評估公眾對服務的看法等目的。

（五）Cookie 和跟蹤技術

我們的網站使用多種技術方法收集和跟蹤設備資訊和使用數據，包括網路信標、Cookie、本地存儲和設備 ID。有關我們如何使用 Cookie 和類似技術以及您如何行使對 Cookie 的選擇權的更多資訊，請參閱我們的《Cookie 政策》。

（六）個性化鏈接和標籤

我們可能會使用個性化鏈接和標籤來跟蹤來自我們網站和電子郵件的鏈接效果。個性化鏈接和標籤是附加在 URL 上的文本片段，幫助我們確定網站流量的來源並跟蹤行銷活動的效果。

（七）驗證碼服務

我們的網站可能會採用穀歌的驗證碼服務，以防止惡意活動。穀歌會收集相關資訊，以確定訪問者是人還是機器人，同時用於一般安全目的。您可以在穀歌的隱私政策和服務條款頁面查閱穀歌收集的資訊及其使用用途。
